# Benjamin Ng Portfolio

Personal portfolio site for Benjamin Ng Yi Ting, focused on software engineering, cybersecurity operations, automation, and selected full-stack projects.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Checks

```bash
npm run lint
npm run build
```

The site is static-friendly and ready to deploy on Vercel with the default Next.js settings.
